## Quick Reference

- **Stack:** Node.js + Express + React 17 + GraphQL + PostgreSQL 16 + TailwindCSS 4
- **Versión:** EverShop 2.1.0
- **Módulos MVP:** auth, base, catalog, checkout, oms, graphql, setting, customer, cod
- **Idioma target:** Español (Guatemala)
- **Moneda target:** GTQ (Quetzal)
- **Método de pago MVP:** COD (Pago Contra Entrega)
- **Deploy:** Docker + nginx + Let's Encrypt
